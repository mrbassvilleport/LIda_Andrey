# Маршруты по Португалии для Лиды и Андрея

Статический сайт с красивой подборкой маршрутов по Лиссабону и другим местам Португалии. Проект сделан как локальная страница для друзей и может публиковаться как обычный `index.html` без сборки.

## Что внутри

- один файл [index.html](./index.html)
- карты на Leaflet + OpenStreetMap
- карточки маршрутов с подробным модальным просмотром
- фотографии локаций через Wikimedia Commons
- адаптивная вёрстка для телефона и десктопа

## Локальный запуск

Откройте `index.html` в браузере.

## Публикация

Проект подходит для:

- GitHub Pages
- Vercel

Для GitHub Pages достаточно запушить репозиторий и в настройках GitHub включить Pages для ветки `main`.
