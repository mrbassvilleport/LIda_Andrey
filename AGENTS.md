# Маршруты по Португалии для Лиды и Андрея

## О проекте
Интерактивная веб-страница с 4-5 маршрутами по Лиссабону и другим городам Португалии. Создаётся для друзей (Лида и Андрей), которые приезжают в гости. Страница — красивая презентация маршрутов на выбор.

## Требования
- **Формат**: Статический сайт (HTML + CSS + JS), один `index.html`
- **Деплой**: Vercel (просто ссылка для друзей)
- **Язык контента**: Русский
- **Контент**: Пользователь сам подбирает места для каждого маршрута и передаёт мне — я наполняю карточки
- **Для создания интерфейса**: Использовать скилл `frontend-design`

## Что должно быть в каждой карточке маршрута
1. Название маршрута и краткое описание
2. Интерактивная карта с маркерами точек и линией маршрута (Leaflet.js + OpenStreetMap)
3. Фотографии ключевых мест (бесплатные источники: Wikimedia Commons, Unsplash)
4. Краткий рассказ о каждом месте
5. Практическая справка: длительность, расстояние, советы
6. Ссылки на источники (Wikipedia, Google Maps)

## Дизайн
- Цветовая гамма: азулежу-синий (#1B4F72) + тёплый песочный (#F5E6CC)
- Карточки с обложкой-фото, кликабельные
- Адаптивный дизайн (мобильный + десктоп)
- Мягкие тени, скруглённые углы, плавные анимации

## Технологии
- Leaflet.js (CDN) — карты
- Vanilla JS — без фреймворков
- Современный CSS (grid, flexbox)

## Рабочий процесс
1. Создать шаблон с одним маршрутом-примером для утверждения дизайна
2. Пользователь присылает места для каждого маршрута
3. Наполняю карточки: описания, координаты, фото
4. Деплой на Vercel
